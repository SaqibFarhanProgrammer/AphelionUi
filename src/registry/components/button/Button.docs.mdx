# Button Component

The `Button` component is a flexible and accessible button built with Tailwind CSS and TypeScript.

## Installation

You can install the button component via the CLI:


npx aphelion-ui add button


## Usage

Import the button component into your project and use it like this:

```tsx
import { Button } from 'aphelion-ui';

export default function Example() {
  return <Button variant="primary" size="md">Click Me</Button>;
}
Props
Prop	Type	Description	Default
variant	`"primary"	"secondary"`	Style variant of the button
size	`"sm"	"md"	"lg"`
disabled	boolean	Disables the button if true	false
onClick	() => void	Callback function when button is clicked	undefined
Variants

The button supports different variants:

primary: Filled button with dark background and light text
secondary: Outlined button with light background
Sizes

Available sizes:

sm: Small button
md: Medium button (default)
lg: Large button
States

The button supports various states:

Default: Normal appearance
Hover: Changes on mouse hover
Focus: Keyboard focus state
Disabled: When the button is inactive
Loading: Show a spinner (if implemented)
Accessibility
The button is keyboard navigable (using Tab).
ARIA attributes are supported.
Screen reader friendly.
Best Practices
Always ensure buttons are used for actions (not links).
Avoid overloading buttons with too many styles.
